# Udta Mukeshhh

A simple, stupid game I made in couple of hours based on the famous **Flappy Bird** game.

Extra feature: \
You can switch to either a traditional limitless arcade (for crushing those High scores !!) \
or \
Switch to standard level of limited number of obstacles.
